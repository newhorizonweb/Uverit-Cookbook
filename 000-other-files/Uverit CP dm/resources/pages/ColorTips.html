Color Tips

  • Do's

  • Use contrasting colors to draw attention. Dark button on a light background seems more important than a light one.
  • Take advantage of opacity / half-transparency. You can use it on hovered buttons or in animations.
  • Experiment! The best way to design is to test what goes hand in hand. It is worth spending some time on this, it will pay off.
  • You should use accent colors to attract to more important information, such as a button to a different page. Be careful not to use it too often.
  • You should test text and background contrast on different devices. Each screen is slightly different. You can see something that others cannot.

  • Don'ts

  • Don't match many dark colors close to eachother. This may create confusion about the information priority.
  • Don't use completly black color for plain text. It's much easier to read when you use dark grey on a light background.
  • You shouldn't use many colors in one place. Pick the most important elements and use it there, so there is no distraction.
  • If you have completly different elements / sections, don't use the same colors for them. Separate them so as not to cause confusion.
  • Don't waste colors for less important things. People should notice something that will encourage them to delve into more of your content.
